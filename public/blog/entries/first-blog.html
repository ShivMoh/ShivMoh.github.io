My First Blog Post

May 18, 2024 5 min read

Welcome to my blog! This is a sample blog post to get you started. You can edit this HTML file to write your own content.

Getting Started

To add a new blog post, you'll need to:

  1. Create a new HTML file in the /public/blog/entries/ directory
  2. Add an entry to the /src/app/blog/entries.json file with metadata
  3. Fill in the blog content in the HTML file

Blog Structure

Each blog post is a standalone HTML file. Feel free to customize the styling and add your own CSS. The main blog page will automatically discover and list your posts based on the metadata in entries.json.

Metadata Fields

When adding a new blog entry to entries.json, use these fields:

  • id - Unique identifier (kebab-case)
  • title - Blog post title
  • date - Publication date (YYYY-MM-DD)
  • excerpt - Short summary for the listing
  • tags - Array of tags for filtering
  • path - Path to the HTML file

Example Entry

Here's what an entry in entries.json looks like:

{
  "id": "my-awesome-post",
  "title": "My Awesome Post",
  "date": "2024-05-18",
  "excerpt": "This is a brief description of my post.",
  "tags": ["tech", "javascript"],
  "path": "/blog/entries/my-awesome-post.html"
}

Styling Tips

The blog entry pages already have a dark theme that matches your site. You can customize the CSS in the <style> section of your HTML file, or add more complex styling as needed.
