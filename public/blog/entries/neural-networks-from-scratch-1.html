Neural Networks from Scratch

So I would've started by following along the book, Neural Networks from scratch by Harrison Kinsley and Daniel Kukiela. It was really good for helping me understand the fundamentals, especially since it took a strong emphasis on showing code rather than delving too deeply into mathematics (tho there was a fair share of that too!). One thing I liked is that it defined concepts from a more pragmatic perspectice. Specifically I liked how it explained the concept of a tensor. Now, mathematically I believe that a tensor is a n by n dimensional structure that is the umbrella term for things like vectors and matrices where a vector is a 1D array of numbers and a matrix is a 2D array of numbers. Now what I like that while a tensor might have certain nuances, in practice, as the book would've put it bluntly, all a programmer needs to really know is that its an array of N dimensions. And that's kind of it.

The Simpliest classifer.

So while going through the book was fun and all (I did not finish it entirely, mind you), at the end of the day building something without following exactly what was there is what of course really helped my understanding. My approach was simple, I had claude make the absolute simpliest network for training a classifer on the MNIST dataset for classifying numbers and attempted to recreate that from what I learnt. The book does cover training a classifer on the MNIST dataset but i found that it introduces a lot of tools that aren't really needed to get the most basic classifer up and running. Things like regularisation and momentum were nice concepts to learn but I always like learning the bare minimum to get something working from there so that's what I did.

The neural network to recreate was as follows (code is pytorch): 

class SimplestMNIST(nn.Module):
    def __init__(self):
        super().__init__()
        self.flatten = nn.Flatten()
        self.dense = nn.Linear(784, 10)

    def forward(self, x):
        x = self.flatten(x)
        return self.dense(x)
